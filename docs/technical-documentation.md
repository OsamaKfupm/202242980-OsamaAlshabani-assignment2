# Technical Documentation

## Project Structure
- `index.html`: Main structure of the website
- `css/styles.css`: Styling and responsive design
- `js/script.js`: JavaScript functionality
- `assets/images`: Images used in the site
- `docs/`: Documentation files

---

## Design Decisions
- Used Flexbox for layout (navigation, sections)
- Used CSS Grid for project cards
- Implemented responsive design using media queries
- Chose simple and clean UI for readability

---

## Features Implementation

### Responsive Design
- Media queries adjust layout for smaller screens
- Grid layout changes from 2 columns to 1 column on mobile

### Theme Toggle
- JavaScript toggles a `dark-mode` class on the body
- CSS changes colors based on the active theme

### Contact Form
- Frontend-only form with validation
- Displays alert on submission

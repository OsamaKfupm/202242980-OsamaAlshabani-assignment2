# Personal Portfolio Website

## 📌 Project Description
This project is a simple, responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases basic web development skills, including layout design, responsiveness, and interactivity.

The website includes sections such as About Me, Projects, and Contact.

---

## 🚀 Features
- Responsive design (desktop, tablet, mobile)
- Navigation bar with smooth layout
- Projects section with cards
- Contact form (frontend only)
- Dark/Light theme toggle (JavaScript)

## 🛠️ Technologies Used

- CSS3 (Flexbox & Grid)
